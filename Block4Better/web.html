Block4Better

"Interruption & Distractions are never welcomed. So what are you waiting for, BLOCK THEM at your ease."

Distractions during work & studying is never welcomed while interruptions in entertainment are always irritating. So just block them. Block4Better chrome extension helps you in blocking the websites along with messages & pop ups sent by those. User can choose & edit the website list to be blocked according to their wish & time.

HOW TO INSTALL

WANT TO EDIT THE WEBSITES TO BE BLOCKED LIST??

Step1 --> Click on the link & download the extension folder

Step2 --> Go to chrome://extensions/

Step3 --> Ensure Developer Mode is on

Step4 --> Click on Load Unpacked & Upload the Extension folder

Step5 --> Ensure the uploaded Block4Better extension is on

>>> Open the Chrome extension folder in the compiler (Block4Better)

>>> Go into the operations.js file

>>> Edit the array named block with urls of website you want to block

>>> And now YOU are READY to Block Set Go...
